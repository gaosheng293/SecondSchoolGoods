# SecondSchoolGoods —— 校园二手交易小程序

**一个功能完整、美观现代、支持全校发布/搜索/点赞的校园二手交易平台**  
基于 UniApp + Spring Boot + MySQL 打造




---

### 核心功能

- 校园专属：自动绑定大学，只看本校商品，交易更安全
- 瀑布流推荐 + 搜索关键词高亮
- 一键发布商品（支持多图、价格、描述、联系方式）
- 分类浏览（数码、服饰、美妆、运动、书籍等）
- 点赞 / 取消赞（实时同步，点赞数维护）
- 商品详情页（查看大图、联系卖家、分享）
- 我的发布、我的点赞、编辑商品
- 切换大学（支持多校切换）
- 搜索历史 + 热门搜索
- 图片上传（本地直传后端）

---

### 技术栈

| 技术        | 版本/说明                          |
|-------------|------------------------------------|
| 前端        | UniApp（Vue3 + Composition API）  |
| 后端        | Spring Boot + Spring Data JPA      |
| 数据库      | MySQL 8                            |
| 图片存储    | 本地磁盘（可改阿里云OSS）           |
| 认证        | JWT Token                          |
| 部署方式    | 微信小程序 + 宝塔/Nginx + Java Jar |
